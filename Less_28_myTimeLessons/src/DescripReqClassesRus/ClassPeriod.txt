******* Class Period - Класс Period *******

public final class Period extends Object
                          implements ChronoPeriod,
                                     Serializable,
                                     TemporalAmount


Класс Period моделирует количество или сумму времени в годах, месяцах и днях.
Основанный на дате период времени в календарной системе ISO-8601, например:
«2 года, 3 месяца и 4 дня».
------------------------------------------------------------------------------
См. Duration для временного эквивалента этого класса.
https://docs.oracle.com/javase/8/docs/api/java/time/Duration.html
------------------------------------------------------------------------------

Продолжительность и периоды различаются обработкой летнего времени при добавлении
в ZonedDateTime. Продолжительность добавит точное количество секунд, поэтому
продолжительность одного дня всегда составляет ровно 24 часа. Напротив, Period
добавит концептуальный день, пытаясь сохранить местное время.

Например, рассмотрите возможность добавления периода в один день и продолжительности
в один день до 18:00 вечера перед переходом на летнее время. Период добавит
концептуальный день и приведет к ZonedDateTime в 18:00 следующего дня. Напротив,
Duration добавит ровно 24 часа, в результате чего ZonedDateTime будет в 19:00 следующего
дня (при условии, что интервал летнего времени составляет один час).

Поддерживаемые единицы периода: ГОДЫ, МЕСЯЦЫ и ДНИ.
Все три поля присутствуют всегда, но могут быть установлены равными нулю.

Календарная система ISO-8601 — это современная гражданская календарная система,
используемая сегодня в большинстве стран мира. Она эквивалентна пролептической
системе григорианского календаря, в которой сегодняшние правила для високосных
лет применяются на все времена.

Period моделируется, как направленное количество времени, что означает, что
отдельные части периода могут быть отрицательными.

Для сравнения объектов класса следует использовать метод equals.
Класс Period является неизменяемым и потокобезопасным.

******* МЕТОДЫ *******

- Temporal addTo(Temporal temporal) - Добавляет этот период к
  указанному временному объекту.

- static Period between(LocalDate startDateInclusive, LocalDate endDateExclusive) -
  Получает Period, состоящий из количества лет, месяцев и дней между двумя датами.

- boolean equals(Object obj) - Проверяет, равен ли этот период другому периоду.

- static Period from(TemporalAmount amount) - Получает экземпляр Period из временной суммы.

- long get(TemporalUnit unit) - Получает значение запрошенной единицы.

- IsoChronology	getChronology() - Получает хронологию этого периода, которая представляет
  собой календарную систему ISO.

- int getDays() - Получает количество дней этого периода.

- int getMonths() - Получает количество месяцев этого периода.

- List<TemporalUnit> getUnits() - Получает набор единиц, поддерживаемых этим периодом.

- int getYears() - Получает количество лет этого периода.

- int hashCode() - Хэш-код для этого периода.

- boolean isNegative() - Проверяет, являются ли какие-либо из трех единиц этого
  периода отрицательными.

- boolean isZero() - Проверяет, равны ли нулю все три единицы этого периода.

- Period minus(TemporalAmount amountToSubtract) - Возвращает копию этого периода
  с вычтенным указанным периодом.

- Period minusDays(long daysToSubtract) - Возвращает копию этого периода с
  вычитанием указанных дней.

- Period minusMonths(long monthsToSubtract) - Возвращает копию этого периода с
  вычитанием указанных месяцев.

- Period minusYears(long yearsToSubtract) - Возвращает копию этого периода с
  вычтенными указанными годами.

- Period multipliedBy(int scalar) - Возвращает новый экземпляр с каждым элементом
  в этом периоде, умноженным на указанный скаляр.

- Period negated() - Возвращает новый экземпляр с инвертированием каждой суммы
  за этот период.

- Period normalized() - Возвращает копию этого периода с нормализованными
  годами и месяцами.

- static Period of(int years, int months, int days) - Получает Period,
  представляющий количество лет, месяцев и дней.

- static Period ofDays(int days) - Получает Period, представляющий количество дней.

- static Period ofMonths(int months) - Получает Period, представляющий число месяцев.

- static Period ofWeeks(int weeks) - Получает Period, представляющий количество недель.

- static Period ofYears(int years) - Получает Period, представляющий число лет.

- static Period parse(CharSequence text) - Получает период из текстовой строки,
  такой как PnYnMnD.

- Period plus(TemporalAmount amountToAdd) - Возвращает копию этого периода с
  добавленным указанным периодом.

- Period plusDays(long daysToAdd) - Возвращает копию этого периода с указанными
  добавленными днями.

- Period plusMonths(long monthsToAdd) - Возвращает копию этого периода с
  добавлением указанных месяцев.

- Period plusYears(long yearsToAdd) - Возвращает копию этого периода с
  добавленными указанными годами.

- Temporal subtractFrom(Temporal temporal) - Вычитает этот период из
  указанного временного объекта.

- String toString() - Выводит этот период в виде строки, например P6Y3M1D.

- long toTotalMonths() - Получает общее количество месяцев в этом периоде.

- Period withDays(int days) - Возвращает копию этого периода с указанным количеством дней.

- Period withMonths(int months) - Возвращает копию этого периода с указанным количеством месяцев.

- Period withYears(int years) - Возвращает копию этого периода с указанным количеством лет.

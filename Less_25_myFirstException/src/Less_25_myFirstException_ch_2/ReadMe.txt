Примеры реализации вариантов исключений.

1. Объявите переменную со значением null. Вызовите метод у этой
переменной. Отловите возникшее исключение.

2. Напишите код, который создаст, а затем отловит
ArrayIndexOfBoundsException.

3. Создайте собственный класс-исключение - наследник класса Exception.
Создайте метод, выбрасывающий это исключение. Вызовите этот метод
и отловите исключение. Вывести stack trace в консоль.

4. Создайте собственный класс-исключение - наследник класса RuntimeException.
Добавьте в конструктор своего класса возможность указания сообщения.

5. Стимулируйте выброс одного из исключений JAVA, отловите его и выбросьте
свое собственное, указав в качестве причины отловленное.

6. Создайте метод случайным образом выбрасывающий одно из 3-х видов
исключений. Вызовите этот метод в блоке try-catch, отлавливающее любое
из трех.
